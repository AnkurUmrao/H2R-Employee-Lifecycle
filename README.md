# H2R-Employee-Lifecycle
SAP HR H2R Lifecycle Management System built with Flask &amp; JSON. Features include Employee Master Data (PA30), Personnel Actions/Hiring (PA40), Payroll Processing with statutory deductions, Time Management (Leave/Absences), Training &amp; Development, and comprehensive HR analytics.
